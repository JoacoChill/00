9-¿Cuál es la estructura básica del HTML?

La estructura HTML de una página web se compone de las etiquetas "html", que indica el comienzo de la página; "head", que contiene información más descriptiva sobre el sitio, a dentro de este se usan otras etiqueticas como: "meta" y "title"; y "body", que conforma todos los elementos visibles de la página

2-¿Qué es el HTML? ¿En qué consiste?

HTML ("Hypertext Markup Language") no es un lenguaje de programación. Es un lenguaje de marcado que le dice a los navegadores web cómo estructurar las páginas web que estás visitando. Consite en estructurar y desplegar una página web y sus contenidos

3-¿Qué es una etiqueta? ¿Funciona si se escribe tanto en mayúscula como en minúscula? ¿Cuál es la mejor práctica para escribir etiquetas?

Las etiquteas son códigos utilizados para marcar el texto de una página web, con el fin de dar instrucciones al navegador sobre cómo mostrarlo
Las etiquetas en HTML no distinguen entre mayúsculas y minúsculas. Así que se pueden escribir tanto en mayúsculas como en minúsculas
La mejor práctica es escribir todas las etiquetas en minúsculas para mantener la coherencia y legibilidad, entre otros motivos.

4-¿A qué llamamos elemento HTML? ¿Cuál es su anatomía? Explicar sus partes.

El elemento "html Es el que encierra todo el contenido de la página entera y, a veces, se le conoce como el elemento raíz
la anatomia de un elemento HTML Establece dónde comienza o empieza a tener efecto el elemento
Sus partes son:
La etiqueta de apertura: Esta etiqueta de apertura marca dónde comienza el elemento o comienza a tener efecto.
contenido: Este es el contenido del elemento.
La etiqueta de cierre: Es lo mismo que la etiqueta de apertura, excepto que incluye una barra diagonal antes del nombre del elemento. Esto indica dónde termina el elemento

11-¿Para qué sirve la etiqueta img? ¿Cuáles son sus atributos? ¿Para qué sirven cada uno de ellos?

La eiqueta "img" sirve para insertar una imagen simple en una página web.
La etiqueta "img" cuenta con dos atributos, el atributo “src” y el segundo atributo es “alt”. El atributo de “src” es para mostrar en dónde se encuentra la imagen que vamos a incrustar, y el atributo "alt" es para realizar un texto que describe una imagen.
